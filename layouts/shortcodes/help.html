Getting Help

If you have any questions about getting things set up or if you’re not sure about how to address a problem with the code or documentation, we’re here to help:

  • For people newer to open source, the gitter chatroom is easily accessible through a web browser and authenticates with Github.
  • If you are working on a project issue, ask for help by commenting containing +status/help-wanted or +status/docs-help on the issue. Commenting like this creates a label others can filter on.
